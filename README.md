# Kaggle Titanic Analysis

## Overview
Kaggle Titanic dataset を用いたデータ解析および機械学習。

## Technologies
- Python
- pandas
- matplotlib
- scikit-learn

## What I Did
- CSVデータ読み込み
- 欠損値処理
- 特徴量エンコーディング
- データ可視化
- RandomForestClassifier による生存予測
- Feature Importance 解析

## Features Used
- Pclass
- Sex
- Age
- Fare
- Embarked

## Result
Accuracy: 約0.80

## What I Learned
- データ前処理の重要性
- 特徴量選択
- 機械学習モデル評価
- Feature Importance の解釈
